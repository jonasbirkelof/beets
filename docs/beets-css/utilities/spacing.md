# Spacing

- The spacers **1-4** are the same as the Bootstrap default values
- **5** is set to the gutter width (30px)
- **6** is the original bootstrap value for **5**
- **7 and 8** are added for more variations

`$spacer` = 1rem = 16px.

|     | Size                 |
| --- | -------------------- |
| 0   | 0                    |
| 1   | `#!scss $spacer * .25`      |
| 2   | `#!scss $spacer * .5`       |
| 3   | `#!scss $spacer`            |
| 4   | `#!scss $spacer * 1.5`      |
| 5   | `#!scss $grid-gutter-width` |
| 6   | `#!scss $spacer * 3`        |
| 7   | `#!scss $spacer * 4`        |
| 8   | `#!scss $spacer * 5`        |
